# Placement Preparation Hub

An AI-powered platform designed to help students prepare for placements through coding practice, aptitude tests, and interview preparation resources.

## Features
- Coding practice modules
- Beginner to Advanced quizzes
- Student dashboard
- Blog section
- Profile management
- Placement resources

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download the repository
2. Open index.html in your browser
